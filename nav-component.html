Atlas Left Nav — reusable component

One file (assets/atlas-nav.js), no build step. Self-injects scoped styles + markup.

Drop-in usage (data attributes)

<div data-atlas-nav
     data-active="bookkeeping/projects"
     data-client="bright-bloom"></div>
<script src="assets/atlas-nav.js"></script>

Or initialise from JS

AtlasNav.init('#nav', {
  active: 'taxation/data',
  client: 'novatech',
  user: { name: 'Sarah K.', role: 'Sr. Bookkeeper', initials: 'SK' },
  hrefs: { 'bookkeeping/data': 'bookkeeping-data.html' },  // override links
});
